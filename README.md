# TugasKu

Platform komuniti tugasan ganjaran & engagement (Next.js + Tailwind + Supabase).
Brand & copy adalah original — struktur/flow direka semula dari kosong.

## Jalankan secara lokal

```bash
npm install
cp .env.example .env.local   # isi kunci Supabase anda
npm run dev
```

Buka http://localhost:3000

## Halaman

| Laluan          | Fungsi                              |
| --------------- | ----------------------------------- |
| `/`             | Landing / sales page                |
| `/marketplace`  | Senarai servis / kempen             |
| `/log-masuk`    | Login (Google + emel)               |
| `/daftar`       | Register                            |
| `/dashboard`    | Wallet, tugasan, kempen, affiliate  |

## Supabase — setup ringkas

1. Cipta projek di https://supabase.com
2. Salin **Project URL** & **anon public key** ke `.env.local`
3. Aktifkan **Google** provider: Authentication → Providers → Google
   (guna OAuth client dari Google Cloud Console; redirect URL Supabase)
4. Jalankan skema di bawah dalam SQL Editor.

### Skema asas

```sql
-- Profil pengguna (dipautokan dari auth.users)
create table public.profiles (
  id uuid primary key references auth.users on delete cascade,
  full_name text,
  wallet_balance numeric default 0,
  ref_code text unique,
  created_at timestamptz default now()
);

-- Kempen (dibeli oleh bisnes)
create table public.campaigns (
  id bigint generated by default as identity primary key,
  owner uuid references auth.users,
  platform text not null,
  service_type text not null,
  target_url text,
  quantity int not null,
  delivered int default 0,
  status text default 'pending',
  created_at timestamptz default now()
);

-- Tugasan (dijana dari kempen, disiapkan oleh pengguna)
create table public.tasks (
  id bigint generated by default as identity primary key,
  campaign_id bigint references public.campaigns on delete cascade,
  reward numeric not null,
  status text default 'open',
  created_at timestamptz default now()
);

-- Penyiapan tugasan
create table public.submissions (
  id bigint generated by default as identity primary key,
  task_id bigint references public.tasks,
  user_id uuid references auth.users,
  proof text,
  status text default 'pending',
  created_at timestamptz default now()
);

-- Pengeluaran wallet
create table public.withdrawals (
  id bigint generated by default as identity primary key,
  user_id uuid references auth.users,
  amount numeric not null,
  method text,
  status text default 'requested',
  created_at timestamptz default now()
);

alter table public.profiles enable row level security;
create policy "own profile" on public.profiles
  for all using (auth.uid() = id);
```

## Deploy ke Vercel

1. Push repo ke GitHub
2. Import ke Vercel
3. Tetapkan env `NEXT_PUBLIC_SUPABASE_URL` & `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. Deploy
